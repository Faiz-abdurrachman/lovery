# 🎨 Lovery Design System

## Pendahuluan

Design System merupakan pedoman resmi dalam merancang seluruh antarmuka Lovery Studio Management System.

Dokumen ini memastikan seluruh halaman memiliki tampilan yang konsisten, mudah digunakan, dan mencerminkan identitas Lovery Photography.

Seluruh desain Figma maupun implementasi Frontend wajib mengikuti aturan pada dokumen ini.

# 18. Dashboard Rules

Dashboard tidak boleh terlalu banyak menggunakan warna pink.

Pink hanya digunakan sebagai:

- Primary Action
- Active State
- Badge penting

Dashboard harus lebih fokus pada keterbacaan data.

# Ringkasan

Design System menjadi pedoman utama seluruh desain Lovery Studio Management System.

Seluruh keputusan visual harus mengacu pada dokumen ini agar identitas brand tetap konsisten, pengalaman pengguna tetap nyaman, dan implementasi frontend menjadi lebih mudah.
